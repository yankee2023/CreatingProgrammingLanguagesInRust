## 2.スタックベース仮想マシン

- unwrap  
エラーハンドリングではなく、プログラムの実行中に予期しないエラーが発生した場合に使用する。  
戻り値がNoneやErrの場合、'panic!'マクロが呼ばれクラッシュする。
